.. image:: ../../../../../figures/scale_selector_configuration.png

YAML-Definition:

.. code-block:: yaml

   tooltip: "Scale" # text to use as tooltip
   target: ~        # Id of Map element to query
   label: false     # false/true to label the scale selector
